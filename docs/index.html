Introduction

This is the site for my home-built 6502 computer system, Búri. More information on why I decided to make Búri, what I hope to achieve and what to expect from this site can be found on the about page.

Hardware

Details on the actual hardware of Búri. Descriptions of the design software used and links to the schematics.

Software

Details on the Operating System and user software for Búri. Descriptions of the non-hardware memory map and guides for writing software.

Peripherals

Add-on hardware modules for Búri such as a serial port, debugging "monitor" system and LCD display.
